POS Tagger Comparison

An Interactive Comparison of Popular POS Taggers

This application translates a static report on Part-of-Speech (POS) taggers into an interactive experience. Instead of just reading, you can explore each tool, see a simulation of its output, and compare their strengths at a glance. The "best" tool depends on the task, from learning (NLTK) to production (spaCy).

Tagger Comparison Chart

Toggle between metrics to see how the tools stack up (based on the report's findings).

Simulates the output of NLTK's `pos_tag` function.

[ ]

Python Code Snippet

>>> import nltk
>>> tokens = nltk.word_tokenize(
    'Automatic Parts of Speech tagging is so cool!')
>>> nltk.pos_tag(tokens)

spaCy

Widely regarded as one of the most powerful and advanced libraries for industrial-strength NLP. It is known for its speed, efficiency, and production-ready models.

Interactive Example

Simulates the rich token output from a spaCy `Doc` object.

Python Code Snippet

import spacy
nlp = spacy.load('en_core_web_sm')
res = nlp(u'Hi, I am Spacy. I can perform
    POS tagging, Tokenization, Lemmatizing,
    Dependency Parsing, Named Entity
    Representation, etc.')
for token in res:
    print(token.text, token.pos_,
        token.tag_)

Stanford POS Tagger

Considered by many to be the most widely used and accurate POS tagger, serving as an academic and research benchmark. Its setup is more complex as it's a Java-based tool.

Interactive Example

Simulates the output of the StanfordTagger NLTK wrapper.

[ ]

Python Code Snippet

from nltk.tag.stanford import StanfordTagger
sample = 'What in the earth are parts
    of speech tags?'
tok = sample.split()
model_file = '/path/to/model.tagger'
model_jar = '/path/to/stanford-postagger.jar'
model_tagger = StanfordTagger(model_file,
    path_to_jar = model_jar,
    encoding = 'UTF-8')
print(model_tagger.tag(tok))

TextBlob

TextBlob is a Python library that offers a simple API for common NLP tasks. It's built on the "giant shoulders of NLTK" and provides a very user-friendly experience, with objects that behave like Python strings.

Interactive Example

Simulates the output of TextBlob's `.tags` property.

[ ]

Python Code Snippet

>>> from textblob import TextBlob
>>> test = TextBlob(
    'TextBlob is also a good parts of speech tagger.')
>>> test.tags

Dependency Parsers (pyStatParser & MaltParser)

These tools go beyond simple tagging. They perform dependency parsing, which analyzes the grammatical structure of a sentence and returns a full parse tree, showing how words relate to each other.

pyStatParser

A simple statistical parser that returns NLTK-compatible parse trees. It uses a CKY algorithm.

MaltParser

A framework for data-driven dependency parsing that can be trained on treebank data.
